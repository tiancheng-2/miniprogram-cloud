# Assets 资源目录

此目录用于存放静态资源文件。

## 缺失的图片资源

以下图片需要手动添加或使用在线URL替代：

- `default-avatar.png` - 默认用户头像
- `default-cover.jpg` - 默认餐厅封面
- `default-dish.jpg` - 默认菜品图片
- `share-cover.png` - 分享封面图
- `marker-red.png` - 地图红色标记点

## 临时解决方案

在开发阶段，可以：
1. 使用微信默认头像（不设置 src）
2. 使用在线占位图服务（如 placehold.co）
3. 使用纯色背景替代
